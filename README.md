# Projektplanung - Beispiel: Meine Portfolio-Webseite

## Projektidee
Ich möchte eine persönliche Portfolio-Webseite erstellen, um meine Projekte und Fähigkeiten im Bereich Webentwicklung vorzustellen. Die Webseite wird ein modernes Design haben und meine bisherigen Arbeiten enthalten.

## Zweck
Meine eigenen Projekte vorstellen und meine Fähigkeiten im Bereich Webentwicklung präsentieren.

## Struktur und Seitenaufbau
Die Webseite besteht aus einer einzigen Seite (Onepager), die in mehrere klar getrennte Abschnitte gegliedert ist. Durch sanfte Scroll-Animationen gelangt man von einem Bereich zum nächsten.
Vollflächiger Startbereich über die gesamte Bildschirmhöhe
Dunkelblauer Hintergrund
In der Mitte:
Mein Name in grosser, moderner Schrift
Darunter eine kurze Berufsbezeichnung, z. B.:
„Webentwickler mit Fokus auf modernes Frontend“
Ein auffälliger türkisfarbener Button mit der Aufschrift:
„Meine Projekte ansehen“.
Der Button leuchtet auf, wenn man mit der Maus drauf geht.
Optional ein dezentes Hintergrundbild oder eine schlichte Illustration.
Weiter unter befindet sich dann der Projektbereich:
Dieser Bereich ist als Galerie aufgebaut.
Heller Hintergrund in Weiss
Überschrift in dunkelblauer Schrift: „Meine Projekte“.
Darunter ein Raster aus Projektkarten.

## Farbkonzept und Design
- **Farben:** Dunkelblau für Hintergründe als Hauptfarben. Sekundärfarbe: Weiss für Textflächen.
- **Schriftarten:** Eine moderne serifenlose Schrift für Fliessenexte (z. B. Open Sans oder Poppins)
- **Designstil:** Klar, minimalistisch, modern; Keine überladenen Effekte; Abgerundete Ecken bei Karten und Buttons; Leichte Schatten für Tiefe

  
## Zeitplanung
- **Woche 1:** Ideensammlung und erste Layout-Entwürfe.
- **Woche 2:** Grundstruktur in HTML und CSS erstellen. Abgabe Planung
- **Woche 3:** JavaScript-Funktionalitäten einbauen.
- **Woche 4:** Feinschliff.
